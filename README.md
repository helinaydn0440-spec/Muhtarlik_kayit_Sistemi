# Muhtarlik_kayit_Sistemi
SQL Server ile geliştirilmiş, ilişkisel veri tabanı modeline sahip Muhtarlık Yönetim Sistemi mimarisi. Proje; iş mantığını yönetmek adına Stored Procedure, Kullanıcı Tanımlı Fonksiyonlar ve View yapılarını barındırır. Ayrıca talep/şikayet süreçlerinde dinamik e-posta bildirimi tetikleyen Trigger mekanizması ve gelişmiş Constraints içerir.
